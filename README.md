# Image Compressor Tool

A modern and fast web-based image compressor built using React. This tool allows users to compress images directly in the browser without uploading to any server.

## 🚀 Features

* Compress images (JPG, PNG, WEBP)
* Preview original and compressed image sizes
* Download compressed image
* Rename image before download
* Choose output format (JPG, PNG, WEBP)
* Clean and responsive UI

## 🛠️ Tech Stack

* React (Frontend)
* JavaScript
* browser-image-compression

## 📸 How to Use

1. Upload an image (JPG, PNG, WEBP)
2. Adjust compression settings
3. Preview compressed result
4. Rename file (optional)
5. Download compressed image
